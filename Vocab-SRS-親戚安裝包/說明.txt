------------------------------------------------------------
建議閱讀順序
------------------------------------------------------------

1. 先看 01-安裝說明 (md 或 docx 任一個都可以,內容一樣)
   - 跟著做 8 個步驟,30–45 分鐘可以裝好
   - 步驟包含:開 GitHub/Firebase 帳號 → Fork → 設 Firestore
     → 改 firebaseConfig → 鎖 API Key → 開 GitHub Pages
     → 手機安裝 PWA

2. 安裝完成後,看 02-使用手冊
   - 怎麼新增小孩使用者
   - 卡片三個按鈕 (忘記/普通/熟悉) 怎麼用
   - 怎麼分享給家人共用
   - 忘記 PIN 怎麼辦

------------------------------------------------------------
關於「字庫/」資料夾
------------------------------------------------------------

裡面是 6,255 字的英文單字資料 (英文、中文翻譯、字典定義、
發音音檔網址),總共約 4.3 MB。

正常情況下你 fork GitHub 程式碼時這份資料就會自動帶過來
(在 docs/data/ 資料夾下),所以「字庫/」只是備份用。

如果哪一天原作者的 GitHub 不見了,你可以把這個資料夾裡的
檔案手動上傳到你 fork 的 docs/data/ 取代。
